# OMB (Oh My Body)

OMB는 사용자가 자신의 운동 활동을 기록하고 관리할 수 있도록 돕는 웹 애플리케이션입니다.<br/>
이 애플리케이션은 운동 종류, 날짜, 시간 등을 기록하는 기능을 제공합니다.

## 기능

- 사용자 인증 (회원가입, 로그인, 로그아웃)
- 운동 기록 추가, 수정, 삭제
- 운동 기록 조회 (리스트 및 상세보기)
- 주간/월간 운동 통계 제공
- 운동 목표 설정 및 달성 확인

## 기술 스택

- **언어**: Java 17
- **프레임워크**: Spring Boot (3.3.5)
- **데이터베이스**: H2 (개발 중) / MySQL (배포 시)
- **프론트엔드**: 미정
- **인증**: Spring Security
- **API 문서화**: Swagger
